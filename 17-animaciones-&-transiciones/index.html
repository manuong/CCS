Animaciones Y Transiciones en CSS

Las animaciones en CSS permiten añadir movimiento y dinamismo a los elementos de una página web. Existen dos principales métodos para crear animaciones en CSS: transiciones y animaciones con @keyframes.


Transiciones

Las transiciones permiten cambiar gradualmente los valores de una propiedad CSS durante un periodo de tiempo específico.
