Pseudoelementos en CSS

Los pseudoelementos en CSS son palabras clave que se añaden a selectores para especificar una parte del elemento seleccionado. Permiten aplicar estilos a partes específicas de un documento que no pueden ser seleccionadas mediante selectores simples.

::before y ::after

::before: Inserta contenido antes del contenido de un elemento.

::after: Inserta contenido después del contenido de un elemento.


::first-letter

Aplica estilos a la primera letra de un bloque de texto.


::first-line

Aplica estilos a la primera línea de un bloque de texto
